# 📝 To-Do List with Deadline Tracking

A simple **Python-based To-Do List application** that helps users manage their daily tasks with deadlines. The application allows users to add, view, update, delete, and track overdue tasks. All tasks are stored in a JSON file, so they remain available even after the program is closed.

## 🚀 Features

* ➕ Add new tasks
* 📋 View all tasks
* ✅ Mark tasks as completed
* 🗑️ Delete tasks
* ⏰ View overdue tasks
* 💾 Save tasks using JSON file handling
* 📂 Automatically load saved tasks when the program starts

## 🛠️ Technologies Used

* Python 3
* JSON
* Datetime Module

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/YourUsername/ToDo_List_Deadline_Tracker.git
```

2. Open the project folder.

3. Run the program:

```bash
python todo.py
```

## 📖 Menu Options

1. Add Task
2. View Tasks

4. Delete Task
5. View Overdue Tasks

## 💾 Data Storage

All tasks are stored in a `tasks.json` file using JSON file handling. This ensures your tasks are saved even after you close the application.
